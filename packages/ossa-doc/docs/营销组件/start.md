# 快速开始

#### 安装

```bash
npm install @ossa/loki
```

#### 单独引入

#### 在 React 应用中使用

```
import BigWheel from '@ossa/loki/components/components/BigWheel';
import '@ossa/loki/dist/style/BigWheel/index.css';

<BigWheel
  config={转盘配置}
  prizeGroup={奖品组配置}
  onPlay={开始函数}
/>
```

#### 在 Taro 应用中使用

```
import BigWheel from '@ossa/loki/components/components/BigWheel';
import '@ossa/loki/dist/style/BigWheel/index.css';

<BigWheel
  config={转盘配置}
  prizeGroup={奖品组配置}
  onPlay={开始函数}
/>
```
